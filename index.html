GoPaint

About GoPaint

GoPaint is a free open source simple painting app inspired by GIMP and Photoshop. It doesn't have all the features—because it's still in beta—but it's lightweight, useful, and growing fast!

If you're a dev and want to improve it seriously, modify the repo on GitHub!
